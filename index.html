Before writing a single line of code, I designed the entire website in Adobe XD. Using Bulma documentation as a frame of reference, each wireframe approximated a style that would eventually be compiled by Bulma and customized with SCSS.

üì≤ Mobile functionality

Particular importance was paid to the app's mobile-responsiveness. Shrinking the Desktop window will automatically display a hamburger menu in the navbar, featuring the same links and routes to pages when viewed fullscreen, except now vertically stacked and left-aligned.

JQuery toggles the "active" class from false to true in order to display the menu and links it contains inside.

AngularJS

First time using Typescript. Routing was easy to set up. Modification of click-handling events and navbar menu class-toggling was straightforward.

To call the API, I installed an injectable service that grabs Observable data, while the FAQ page subscribed to a function which sliced the first 10 entries. One neat feature to Angular is each component had its own scoped SCSS.
